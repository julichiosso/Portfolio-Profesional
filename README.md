# Portfolio Profesional - Julian Chiosso

Este es el portafolio profesional de Julian Chiosso, desarrollado con tecnologías web modernas para destacar habilidades en Backend y Full Stack Development.

## Tecnologías

- **HTML5 & CSS3**: Estructura semántica y estilos avanzados (Glassmorphism, Animaciones).
- **JavaScript (ES6+)**: Lógica de interacción y efectos.
- **Three.js**: Fondo interactivo 3D (partículas/red neuronal).
- **Font Awesome**: Iconografía.
- **Google Fonts**: Tipografías 'Outfit' y 'Space Grotesk'.

## Estructura del Proyecto

- `index.html`: Estructura principal.
- `style.css`: Estilos globales y responsivos.
- `script.js`: Lógica de animaciones y Three.js.
- `profile.jpg`: Imagen de perfil.

## Cómo Desplegar en Vercel

Para publicar este portafolio en internet usando Vercel, sigue estos pasos:

### Opción 1: Desde la Web de Vercel (Recomendado si usas GitHub)

1.  Sube este proyecto a un repositorio en tu cuenta de **GitHub**.
2.  Ve a [vercel.com](https://vercel.com) e inicia sesión.
3.  Haz clic en **"Add New..."** -> **"Project"**.
4.  Importa el repositorio de GitHub que acabas de crear.
5.  En la configuración del proyecto, Vercel detectará automáticamente que es un sitio estático.
6.  Haz clic en **"Deploy"**.

### Opción 2: Usando Vercel CLI (Desde tu computadora)

1.  Asegúrate de tener Node.js instalado.
2.  Instala Vercel CLI globalmente:
    ```bash
    npm i -g vercel
    ```
3.  Abre una terminal en la carpeta de este proyecto.
4.  Ejecuta el comando:
    ```bash
    vercel
    ```
5.  Sigue las instrucciones en pantalla (login, confirmar proyecto, etc.). Usa las opciones por defecto.

¡Listo! Tu portafolio estará online en segundos.
